李嗣涔（1952年8月13日－），台湾高雄县冈山镇（今高雄市冈山区）出生[1]，原籍河南济源，电机工程学学者，主要研究半导体领域，为台湾早期研究非晶硅的学者，亦为国际电机电子工程学会会士（IEEE Fellow），在该领域有一定声望。后来应台湾国家科学委员会主委陈履安等人的邀请转入气功、特异功能方面的研究，而使其学术地位充满争议性。

关于李嗣涔

李嗣涔的特异功能研究涉及到可以手指识字的中日混血女学生高桥舞[27]。在一次TBS电视台主办的节目中，美国魔术师和超自然现象调查员詹姆斯·兰迪认为高桥舞于实验中作弊[28]。尽管詹姆斯·兰迪提出了质疑，但TBS电视台最终的播放仍然舍弃失败的部分，以高桥舞的完美测字为结局[29]。另一次在美国也被抓到以魔术手法作弊[30]。李嗣涔说，手指识字研究也曾经过不同学者验证。1999年，国家同步辐射研究中心主任陈建德等十多名台大、清大及中研院物理学者曾受邀”参观”高桥舞手指识字的过程，但不得检视实验者自备的道具，来宾被安排的观察位置(即观察角度)亦不许调整。他和陈建德合作的手指识字论文，曾刊登在“国际照护期刊”[来源请求]；他和电机系另一名教授[谁？]用核磁共振摄影，测量手指识字时脑部的功能变化，也在全球最重要的功能性核磁共振国际会议上发表论文，证明研究绝非“伪科学”。李嗣涔表示，高桥舞接受别的物理系教授测试时也曾失败，主因在于人是情绪的动物，一旦包装纸条等测试方式改变，都要经适应期才会恢复原有能力。[31]陈建德则指出，他见证高桥舞的手指识字成功率超过九成五，物理系教授不应因她失败过就全盘否认；任何人都不应以实验没有百分之百成功，就完全否定实验成果，否则人类科学不会进步。[31]

自2004年起，李嗣涔在气功、特异功能与信息场的研究方面认为信息场与俄国人理论物理学家所描述过的挠场性质相近，而于近期往这方向研究中，以水晶及功能人进行挠场的发射与接收等测试[来源请求]。有过论文与书籍，认为挠场是有争议性的研究，其研究者的学术地位亦为人所议论。而认同挠场理论的李嗣涔在台湾大学校长遴选时也造成一定的风波。[来源请求] 1998年俄国科学院成立了一个对抗伪科学研究的特别委员会，其领导人Edward P. Kruglyakov博士曾发表了一系列文章，证明Shipov博士所做的挠场研究为伪科学。[32]此外，在杨信男登于2007年3月份科学月刊的信件指出：“李嗣涔坦承他不清楚Shipov的理论，但一直坚信Shipov的扭场就是他自创的信息场，可是却从不提及俄国科学院对Shipov扭场理论调查的结果，对俄国科学院指出其理论谬误视而不见，明确违反了科学研究者应有的基本态度，和公正原则中的“避免因主观立场影响研究结论””。

挠场部分的研究，李教授之研究生论文封锁十年事件，台大物理系杨信男教授指出：“因此，我到台大总图书馆馆藏的‘台大博硕士论文数据库’，希望能影印一份，以便仔细拜读研究。讵料，在我告知系上同仁此想法次日，李嗣涔便越过一般的程序，直接以电话要求台大图书馆锁住该论文，在10年内不许任何人参阅”违反学术伦理[33]。

摘录一：中华大地上的道统传承

200 4年3 月我们 向T 小姐的师父提出要 求 .

“ 请开 示: 师父您答应要 带T 小姐去看月球上的外星 人, 现在可以去看 吗? 或是先去看看外星人的基 地? “

师父回 答 :

”基 地 first,but not toda y。 “

于是第二天我们继续请敦师 父 :

“ 请开 示: 师父请 带T 小姐去看月球上外星人的基 地, 并在月球的地图上指出它的位置。“

这 时T 小姐天眼中看到远方有一大块一大块深土色的建 筑, 非常巨 大, 接下来一切都消失了。我赶紧把月球面向地球一面的地图打开 给T 小姐 看, 上面有数值标示不同的地区。师父回 答.” 80,not exactly.” 8 0 指的是月球中北方圆形的柏拉图陨石坑 , “在 8 0 的东南西北哪个方 向? “ ” 8 0 的西北方向。“我于是在地图上怕拉图陨石坑西北方点了一 点, ”在地表之下 吗? “师父点头。

在 200 3年3 月我们曾问 了T 小姐的师 父 :

”请开 示: 在太阳系中还有哪些行星有智慧的生命存 在 ?(1 ) 水 星 :(2 ) 金 星 :(3 ) 火 星 :(4 ) 木 星 :(5 ) 土 星 :(6 ) 天王 星 :(7 ) 海王星。它们的科技层次比我们高 吗? “

师父回 答 :

” (3 ) 。“

火星上有高等智慧的生 物, 与一般的猜想类似。他们的科技是否比我们 高, 师父没有回答 。 200 4年3 月再 问 :

”请开 示: 火星智慧的生 物, 他们的科 技 (1 ) 已能建造飞行 器, 利用信息场到宇宙旅 行 :(2 ) 与地球类 似 :(3 ) 比地球落 后, 还未有电子科技。“

师父回 答 ..

“ (1 ) 。”

火星人的科技突破时空障碍已经可以进入信息 场, 后来师父又告诉我们火星人也住在地底下。这个火星人科技层次的问 题, 我也请教了孙储琳的师 父, 得到的答案是类似 的 :

“火星有过智慧生 物, 他们已经掌握了高级宇航科 技, 能够进入信息场遨游宇宙。”

孙女士并不确定这些生物是外来的还是本地的。

摘录三十一：孙储琳女的外星师父

200 4年5 月 底, 我再度到北京访问沈教授及孙女 士, 也再一次请教孙女 士,师父所居住的星球是在哪个星座的哪一恒星系 内? 这次我准备的实验器材比较丰 富,首先我给她看一张我从台北天文台买的标准恒星 图, 是以地球为中心所看到的天空恒星分 布, 都是本银河系内相当亮的恒 星, 标示着各个星 座, 还有一些银河系及银河团的位置。

孙女士开了天 眼, 在外星师父的带领 下, 经过了室女座、狮子 座, 最后到了后发座的位 置, 约在赤 经 1 2时 3 0 分、赤北 纬 2 8 度附近的一颗星。
